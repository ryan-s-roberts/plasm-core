**Run Plasm** (live execution) with the same **`logical_session_ref`**, **`program`**, and optional **`reasoning`** as **`plasm`**. Call **only after** reviewing the **`plasm`** dry-run plan. This tool performs real HTTP/API work and may return **`resource_link`** blocks and `_meta.plasm` snapshot references.
**Programs:** use the same newline and heredoc contract as **`plasm`** (initialize workflow — literal U+000A between physical lines; never space-separate multiple bindings on one line). Surface bindings may use **`ident.<relation>`** continuation like **`plasm`**. Bare comma-separated **multi-root** final expressions are supported here (not on **`plasm`**, which stays one composed surface for dry-run).
