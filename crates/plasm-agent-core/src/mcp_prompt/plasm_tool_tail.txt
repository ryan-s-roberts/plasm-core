Prefer **one `program` per call**: use **newlines** for bindings (`ident = …`), then **comma-separated roots** on the final line when returning multiple surfaces (e.g. `sorted, count`). Bracket-render yields **`binding.content`** for string slots. Postfix (`.limit`, `.sort`, `.aggregate`, `.group_by`, `[…]`, `[fields] <<TAG…TAG`) chains when legal.

**Brace `{…}` predicates:** entity-ref slots expect teaching-table constructors, scalar ids, or **`anchor.<relation>`**.

**Continuation:** `label.<relation>` from surface/relation rows or row-preserving projections; aggregate/render/derive labels are not row anchors.
